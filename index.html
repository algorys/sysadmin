Bienvenue

Admitoring propose de la documentation et des tutoriels pour établir une administration et un monitoring de votre parc informatique. Il aura pour but de réunir et de centraliser les informations nécessaire aux administrateurs réseaux et confrères.

Les Serveurs :

Vous allez pouvoir retrouver principalement des tutoriels sur comment installer, configurer et relier les projets suivants :

  • GLPI : Serveur de gestion de parc.

    • GLPI est un logiciel libre de gestion de parc. Il permet d'inventorier celui-ci du simple téléphone, au serveur, en passant par les imprimantes jusqu'aux terminaux finaux. Bref tout ce qui est connecté !

  • FusionInventory : fournit un inventaire de votre parc de manière automatique.

    • FusionInventory permet, grâce à des agents légers, d'inventorier votre parc informatique. Selon vos horaires, désirs, envies... il rappatrie le maximum dans GLPI automatiquement.

  • Shinken : framework de monitoring en python utilisant les plugins nagios.

    Shinken permet de monitorer vos ressources, serveurs, services, ... Il peut fonctionner indépendamment de GLPI ou non et vous signale toute anomalie.

  • Alignak : solution de monitoring en python.

    • Alignak permet de monitorer vos ressources, serveurs, services, ... Peut fonctionner indépendamment de GLPI. Son but principal est la stabilité.

  • Glpi-monitoring : permet d'afficher les données d'un serveur de monitoring dans GLPI.

    Ce plugin pour GLPI permet d'afficher des vues graphique des données d'un serveur de monitoring. Il permet notamment de créer / ajouter vos hôtes à partir de GLPI.

Nous essaieront également de proposer des tutoriels aussi bien en anglais qu'en français.

Il y aura notamment des articles sur les dernières releases des différents projets, sur leurs fonctionnalités futures, d'autres serveurs ou outils connexes, etc.
